Описание проекта

Charmed Books представляет собой платформу электронных книг с интерактивными элементами и анимациями.

Цели проекта

  • Создание современной платформы для чтения.
  • Повышение интереса к электронным книгам.
  • Изучение веб-технологий.

Технологии

  • HTML5
  • CSS3
  • JavaScript
  • Git
  • Markdown
